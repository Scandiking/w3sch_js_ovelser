JavaScript Statements

I HTML-dokumenter kjøres JavaScript statements av nettleseren din.

JavaScript statements består av verdier, operatorer, uttrykk, nøkkelord og kommentarer. Eller på engelsk: values, operators, expressions, keywords og comments.

Statementet "document.getElementById("identifikatornavn").innerHTML = "Hei, sveis" forteller nettleseren din at den skal skrive "Hei, Sveis" inni et element med en identitet vi har gitt elementet. I dette tilfellet "identifikatornavn".

Semikolon: «;»

let a, b, c;
a = 5;
b = 6;
c = a + b;

Semikolon skiller JavaScript statements fra hverandre. Vi legger til et semikolon på slutten av hvert kjørbare statement.

Det første statementet deklarerer tre variabler. På første linje har de ikke fått tilordnet noen verdi

a = 5; tilordner verdien 5 til variabelen a. Samme med b = 6;. Variabelen c blir tilordnet summen av verdien av variabel a og variabel b.

JavaScript White Space

JavaScript ignorerer flere mellomrom. Du kan legge til hvitt rom for å gjøre scriptet ditt mer lesbart. For eksempel er følgende linjer like hva JavaScript angår:

let person = "Hege";
let person="Hege";

Derfor er det ansett som god praksis å bruke mellomrom rundt operatorer. Det er mer behagelig å lese ( (2**2 + 3) / ( 4**3) ) fremfor ((2**2+3)/(4**3)).

JavaScript nøkkelord (keywords)

JavaScript statements starter ofte med et nøkkelord for å identifisere JavaScript-handlingen som skal utføres.

  • var
    • Deklarerer en variabel
  • let
    • Deklarerer en blokkvariabel
  • const
      Deklarerer en blokk-konstant
  • if
      Markerer en block med statements som kjøres avhengig av tilstand
  • switch
      Markerer en blokk med statements som kjøres i forskjellige sammenhenger (cases)
  • for
      Markerer en blokk med statements som kjøres i løkke (loop)
  • function
      Deklarerer en funksjon
  • return
      Går ut av en funksjon
  • try
      Implementerer feilbehandling for en blokk med statements

JavaScript Syntakst

JavaScript Syntax

//How to create variables
var x;
let y;

//How to use variables
x = 5;
y = 6;
let z = x + y;

JavaScript syntakst definerer to typer verdier. Det er faste verdier (fixed values) og variabler (variable values). Faste verdier kalles literals og variable verdier kalles variables.

JavaScript Literals

De to viktigste syntakstreglene for faste verdier er:

  • Tall skrives med eller uten desimaler.
  • Tekststrenger (strings) skrives inni enkelte eller doble anførselstegn som «'» og «"».

JavaScript Operators

JavaScript bruker operatorene til de fire regneartene for å beregne verdier.

console.log(5+10) // gir «15» i konsollen.
a = 5;
b = 10;
c = a + b;
console.log(c) // gir også «15» i konsollen.

JavaScript bruker tilordningsoperatoren «=» for å tilordne verdier til variabler.

let x, y;
x = 5;
y = 6;

JavaScript Expressions

Et uttrykk er en kombinasjon av verdier, variables og operatorer som beregner en verdi. Beregningen kalles en evaluasjon. For eksempel, 5 * 10 evaluerer til 50. Uttrykk kan også inneholde variabelverdier som x * 10.

Verdiene kan også være forskjellige datatyper, sånn som tall og tekststrenger. For eksempel blir "John" + " " + "Doe" evaluert til "John Doe".

JavaScript Keywords

Nøkkelord, eller keywords brukes for å sjekke etter handlinger som skal utføres. let-nøkkelordet brukes for å fortelle nettleseren at den skal lage variabler. Det kan hende at du ser "var" i gammel kode. Ikke bruk "var" i din egen kode, men dersom du ser "var" i andres ferdigskrevne kode, la det være som det er. var er utdatert.

JavaScript Comments

Det er ikke alle JS-statements som eksekveres (kjøres). Kode etter doble skråstreker eller skråtrek stjerne og stjerne skråstrek blir forstått som en kommentar. Kommentarer ignoreres av nettleseren og kommer ikke til å bli eksekvert. Kommentarer er til for utviklere og mennesker.

JavaScript Identifiers/Names

Identifikatorer er JavaScript-navn. Identifikatorer brukes for å navngi variabler, nøkkelord og funksjoner. Reglene for lovlige navn er de samme som i de fleste programmeringsspråk. Et JavaScript-navn må begynne med:

  • En bokstav (A-Z eller a-z, ingen æ, ø, å)
  • Et dollartegn $
  • En understrek _

Påfølgende tegn etter disse kan være bokstaver, sifre, understreker eller dollartegn. Merk at tall ikke tillates som første tegn i navn. På denne måten kan JavaScript lett skille identifikatorer (identifiers) fra tall.
