JavaScript Statements

I HTML-dokumenter kjøres JavaScript statements av nettleseren din.

JavaScript statements består av verdier, operatorer, uttrykk, nøkkelord og kommentarer. Eller på engelsk: values, operators, expressions, keywords og comments.

Statementet "document.getElementById("identifikatornavn").innerHTML = "Hei, sveis" forteller nettleseren din at den skal skrive "Hei, Sveis" inni et element med en identitet vi har gitt elementet. I dette tilfellet "identifikatornavn".

Semikolon: «;»

let a, b, c;
a = 5;
b = 6;
c = a + b;

Semikolon skiller JavaScript statements fra hverandre. Vi legger til et semikolon på slutten av hvert kjørbare statement.

Det første statementet deklarerer tre variabler. På første linje har de ikke fått tilordnet noen verdi

a = 5; tilordner verdien 5 til variabelen a. Samme med b = 6;. Variabelen c blir tilordnet summen av verdien av variabel a og variabel b.

JavaScript White Space

JavaScript ignorerer flere mellomrom. Du kan legge til hvitt rom for å gjøre scriptet ditt mer lesbart. For eksempel er følgende linjer like hva JavaScript angår:

let person = "Hege";
let person="Hege";

Derfor er det ansett som god praksis å bruke mellomrom rundt operatorer. Det er mer behagelig å lese ( (2**2 + 3) / ( 4**3) ) fremfor ((2**2+3)/(4**3)).

JavaScript nøkkelord (keywords)

JavaScript statements starter ofte med et nøkkelord for å identifisere JavaScript-handlingen som skal utføres.

  • var
    • Deklarerer en variabel
  • let
    • Deklarerer en blokkvariabel
  • const
      Deklarerer en blokk-konstant
  • if
      Markerer en block med statements som kjøres avhengig av tilstand
  • switch
      Markerer en blokk med statements som kjøres i forskjellige sammenhenger (cases)
  • for
      Markerer en blokk med statements som kjøres i løkke (loop)
  • function
      Deklarerer en funksjon
  • return
      Går ut av en funksjon
  • try
      Implementerer feilbehandling for en blokk med statements
